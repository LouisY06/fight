# Mecha-Mime: Phygital Combat

Gesture-driven first-person mecha cockpit. Your body controls the mecha via webcam pose tracking.

## Quick Start

```bash
npm install
npm run dev
```

Open the browser and allow camera access. Move your body to control the mecha arms.

## Tech Stack

- **Three.js** — 3D rendering
- **MediaPipe Pose** — real-time body tracking (33 landmarks)
- **Vite** — build tool
